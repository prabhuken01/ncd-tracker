🆘 NEED HELP?
════════════════════════════════════════════════════════════════════════════════

Issue: Files not found
   → Check download location
   → Verify all folders copied (constants, data, documents, ui, utils)

Issue: Deployment fails
   → Check logs in Streamlit Cloud dashboard
   → Verify requirements.txt is present
   → Ensure app.py is in root directory

Issue: App crashes
   → View error logs in Streamlit Cloud
   → Test locally first: streamlit run app.py
   → Run verification: python verify.py

Issue: Data not persisting
   → Expected on Streamlit Cloud (ephemeral storage)
   → Implement Google Drive integration (see deployment guide)
   → Or commit Excel file to GitHub
